    Introduction

    The UnifiedOS: Small Business Server Project is a collaborative open source project to create a GNU/Linux operating system so that small businesses, non-profit organizations, and home users can have a simple server operating system. Also UnifiedOS will be built with software and tools to help businesses to grow it's IT infrastructure as the business grows along with it.

    Features

    Unified Administration

    Configure and view your server through a web browser from a desktop or mobile device.

    Unified Reliability

    UnifiedOS is built upon the Linux kernel, the same kernel that runs the New York Stock Exchange to the World Wide Web.

    Unified Expandability

    As your business grows, your infrastructure will grow too. That's why UnifiedOS has tools to help you expand and configure as you add more servers.

    Unified Business

    Launch powerful appliances such as ownCloud, Collabtive, and sugarCRM on KVM (Kernel Virtual Machine) right out of the box.
