    Introduction

    The unifiedOS: Small Business Server Project is a collaborative open source project to create a GNU/Linux operating system so that small businesses can have a easy to use server operating system on one piece of hardware. Also unifiedOS will be built with software and tools to help businesses to grow it's IT infrastructure as the business grows along with it.

    Tentative Feature List

    • Basic services such as Email, Virtual Private Networking, File Sharing, Backup, Print Sharing, and Collaboration.
    • A unified dashboard for the
    • Tools and Guides to help administer and monitor the server even if the user has little to no experince in IT.
    • Deployment tool for other computer systems such as Kiosks, Desktops, Embedded, and Point-of-Sale systems.
    • Expansion tools to build on additonal server hardware.
